# Movie Website

This project is a movie website created using jQuery, Bootstrap, HTML, CSS, JavaScript, and the TMDb (The Movie Database) API.


### Requirements

- Web Browser (Google Chrome, Mozilla Firefox, etc.)
- Internet Connection
